Servlet+JDBC+Mysql实现用户登录功能示例

发表于 2019-03-28 | 分类于 java

一、用户登录需求

  1. 登录页面，有用户名、密码两个输入框和登录按钮
  2. 点击登录跳转
  3. 登录成功跳转到成功页面，显示用户欢迎信息
  4. 登录失败跳转到失败页面，显示错误信息

二、需求分析

  1. login.html，
    1. username，password输入框
    2. 登录按钮
  2. mysql建立user表
  3. 使用Druid数据库连接池，jdbcTemplate操作数据库
  4. Servlet
    1. 获取用户名和密码
    2. 验证登录
    3. 成功跳转到Success页面
    4. 失败跳转到error页面
